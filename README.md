# Task Manager (Full-Stack)

A simple **Task Manager** application with a **React frontend** and a **Node.js + Express backend**.  

## Main Features
- Create tasks with:
  - Title
  - Description
  - Priority (low / medium / high)
- Task statuses:
  - `active`
  - `pending`
  - `completed`
- Filter tasks by status
- Edit and delete tasks
- Edit task details using a modal



When a task is created in **automatic mode**, its initial status is set to **`active`** by default.

## Running the Backend
cd backend
npm install
npm start

The backend server will run on:
http://localhost:4000

## Running the Frontend
cd frontend
npm install
npm start

The frontend application will open at:
http://localhost:3000

## Technologies Used
Frontend: React, JavaScript, CSS
Backend: Node.js, Express
State Management: React Hooks
Communication: REST API (JSON)
